## 📊 领域分述

### 🤖 AI & Agent
- **SAGE: Multi‑Agent Self‑Evolution for LLM Reasoning**（arXiv:2603.15371，2026‑03‑17）  
  四个Agent（挑战者、规划者、求解者、批判者）从同一LLM骨干中共演化，在数学与代码生成任务上显著提升基线模型（如Qwen‑2.5‑7B）性能。  
  **专家评价**：该工作实现了多智能体系统的“内卷式”自我优化，为无需人工干预的持续能力进化提供了可行路径，有望成为未来自主科研的核心引擎。

- **Brain‑Inspired Graph Multi‑Agent Systems for LLM Reasoning (BIGMAS)**（arXiv:2603.15255，2026‑03‑17）  
  将专用LLM Agent组织为动态图中的节点，通过中央共享工作区协调，在复杂推理任务上持续超越现有多智能体基线。  
  **专家评价**：受脑网络启发的图结构协调机制，为多智能体系统的可扩展性与任务专精分工提供了新的架构思路，尤其适合需要异构专家协作的场景。

### 🧠 大模型 (LLM) & 多模态
- **TriFusion‑LLM: Prior‑Guided Multimodal Fusion with LLM Arbitration for Fine‑grained Code Clone Detection**（arXiv:2603.15004，2026‑03‑16/17）  
  融合启发式相似性先验、AST结构信号与CodeBERT深度语义嵌入，并通过LLM仲裁器在决策边际进行自适应消歧，将BigCloneBench七分类Macro‑F1从0.695提升至0.875。  
  **专家评价**：首次在代码克隆检测中实现多模态信息的“认知级”融合，LLM作为仲裁器有效缓解了语义漂移问题，为其他需要高精度细粒度判别的任务（如医疗图像‑文本匹配）提供了借鉴。

- **Scalable Multilingual Multimodal Machine Translation with Speech‑Text Fusion**（arXiv:2602.21646，2026‑02‑25）  
  提出语音引导的机器翻译框架，将语音与文本作为融合输入注入MLLM，提升翻译质量。  
  **专家评价**：语音‑文本的多模态融合在低资源语种翻译中展现潜力，为打破语言障碍提供了更自然的交互方式。

### 🦾 具身智能 (Embodied AI)
- **InterMimic: Universal Whole‑Body Control for Physics‑Based Human‑Object Interactions**（arXiv:2502.20390v2，2026‑02）  
  从大量不完美运动捕捉数据中学习单一策略，实现多样全身‑物体交互的鲁棒控制，重点解决人与物体间的**不变动力学**问题。  
  **专家评价**：该框架将“模仿学习”推广到动态物体交互场景，为机器人执行诸如搬运、装配等非结构化任务提供了通用控制基底，显著缩短了仿真到现实的适配周期。

- **Embodied Tree of Thoughts (EToT): Real2Sim2Real Planning with Physics‑Based Interactive Digital Twin**（arXiv:2512.08188，2025‑12）  
  利用物理交互数字孪生作为具身世界模型，将操纵规划建模为树搜索，并通过VLM在仿真中诊断失败，实现刚性体动力学与碰撞约束的遵从。  
  **专家评价**：EToT将大模型的“思维树”推理框架具身化，实现了高层规划与物理仿真的闭环验证，为复杂机械操作（如航天器维护、工业组装）提供了可解释、可调试的自主决策管道。
